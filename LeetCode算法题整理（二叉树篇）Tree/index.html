257. Binary Tree Paths

打印二叉树从根节点到叶子节点全部路径。原题

  • iteratively。思路：采用前序遍历二叉树，使用tuple保存节点当前路径，如果是叶子节点，则添加到结果中。开始老是想着用'->'.join()，这样反而麻烦，直接使用字符串保存就好。

  • recursively。参考了StefanPochmann大神的方法。最开始想到一半，中间那层循环想到了，但没想到用递归。
